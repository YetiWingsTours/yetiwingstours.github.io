**Yeti Wings** is a modern, responsive travel agency website based in Nepal. It showcases Himalayan treks, international tours, adventure activities, and travel services. The site features dynamic content filtering, modals with itineraries, AJAX contact forms, and a suite of UI enhancements for better user experience.
